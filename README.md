# 🎬 Infiniti digYtech Ztreaming

**Infiniti digYtech Ztreaming** adalah sebuah platform *streaming micro-drama* premium yang dirancang dengan antarmuka sinematik, elegan, dan sangat responsif. Proyek ini dibangun dengan arsitektur modern untuk memberikan pengalaman menonton layaknya layanan *streaming* kelas dunia.

🚀 **Lihat versi live-nya di sini:** https://iyztreaming.vercel.app/

---

## ✨ Fitur Utama
* **Cinematic UI/UX:** Desain *Dark Mode* murni ala Netflix yang nyaman di mata.
* **Mobile-First Design:** Tampilan yang menyesuaikan secara sempurna di layar *smartphone*, tablet, maupun desktop.
* **Cloud Video Streaming:** Pemutaran video berkinerja tinggi yang terintegrasi langsung dengan Supabase Storage.
* **Modern Stack:** Dibangun menggunakan teknologi *frontend* terbaru untuk kecepatan maksimal.

## 🛠️ The "Silentium" Tech Stack
Proyek ini ditenagai oleh teknologi berikut:
* **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Database & Storage:** [Supabase](https://supabase.com/)
* **Deployment:** [Vercel](https://vercel.com/)
